### Common Verse Patterns

**Pattern 1:** Device with player interaction
\`\`\`verse
using { /Fortnite.com/Devices }
using { /Verse.org/Simulation }
using { /UnrealEngine.com/Temporary/Diagnostics }

my_device := class(creative_device):
    @editable TriggerZone : trigger_device = trigger_device{}
    
    OnBegin<override>()<suspends> : void =
        TriggerZone.TriggeredEvent.Subscribe(OnPlayerEntered)
    
    OnPlayerEntered(Player : ?agent) : void =
        if (ValidPlayer := Player?):
            Print("Player entered!")
\`\`\`

\`\`\`verse

## Key Concepts

### Verse Language Fundamentals
- **Strong typing** with type inference
- **Failure context** (\`<decides>\`, \`<transacts>\`) for safe operations
- **Concurrency** via \`<suspends>\` and async/await patterns
- **Effect specifiers** control function behavior

### UEFN Architecture
- **Devices** are the primary interaction units
- **creative_device** is the base class for custom logic
- **@editable** exposes properties to UEFN editor
- **Events** use subscription pattern (\`.Subscribe()\`)
