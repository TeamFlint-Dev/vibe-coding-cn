# 设计技能库 (Design Skills)

这里存放可复用的设计技能和方法。每个子目录代表一个设计技能领域。

---

## 目录结构

每个技能一个子目录：

```
skills/
├── {skillName}/
│   ├── SKILL.md           # 技能说明和使用方法
│   ├── patterns/          # 该技能的设计模式（可选）
│   ├── examples/          # 使用示例（可选）
│   └── README.md          # 技能简介
```

## 技能 vs 产品

| 类型 | 说明 | 位置 |
|------|------|------|
| **技能** | 设计方法和模式 | `skills/` |
| **产品** | 具体的设计方案 | `products/` |

技能是可复用的方法论，产品是应用技能产出的具体设计。
