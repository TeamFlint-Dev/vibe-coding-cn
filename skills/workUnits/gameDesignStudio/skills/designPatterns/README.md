# 设计模式库 (Design Patterns)

这里存放已验证的、可复用的设计模式。

---

## 模式格式

每个模式一个 Markdown 文件：

```markdown
# [模式名称]

## 适用场景
[描述什么情况下使用这个模式]

## 目标感受
[这个模式要传递什么感受]

## 实现方式
[具体如何实现]

## 注意事项
[需要注意的问题和边界情况]

## 案例
[实际使用案例]
```

1. 从成功的设计中提取

3. 包含至少一个真实案例
4. 更新本 README 的索引表
