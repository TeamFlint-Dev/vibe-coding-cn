# 细节打磨 (Detail Polish)

如何识别和修复出戏点，打磨细节确保体验无瑕。

---

## 核心方法

1. **出戏点识别**: 系统化发现让玩家出戏的瑕疵
2. **场景推演**: 通过场景测试发现边界问题
3. **打磨清单**: 结构化的细节检查方法

---

## 技能产出

- 场景推演模板库
- 常见瑕疵及修复方法
