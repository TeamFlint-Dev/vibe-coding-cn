# IDLE-F-01: "发现钓点"氛围与仪式设计

> **任务ID**: IDLE-F-01  
> **任务名称**: 定义"发现钓点"的感受流程  
> **优先级**: P0  
> **当前阶段**: Phase 2 - 氛围与仪式设计  
> **创建时间**: 2026-01-13

---

## Phase 2: 氛围与仪式设计

基于 Phase 1 定义的三阶段感受流程（好奇 20% → 靠近 20-60% → 确认 100%），本阶段设计具体的视觉、听觉、交互元素来实现这些感受。

---

## 阶段 1: 好奇阶段的氛围营造

### 视觉设计（权重 60%）

#### 核心视觉元素

**1. 水面反光效果**
- **类型**: 动态光点粒子
- **颜色**: 浅蓝色 (#87CEEB) 带微弱白色高光
- **大小**: 2-5 像素，随机闪烁
- **分布**: 钓点中心 3 米半径内，密度 5-8 个光点
- **动画**: 0.5-1.5秒呼吸循环，透明度 30%-70%
- **触发距离**: 15-30 米可见

**2. 鱼影暗示**
- **类型**: 半透明剪影
- **颜色**: 深蓝灰色 (#2F4F4F)，透明度 40%
- **大小**: 根据鱼类大小 10-30 厘米（游戏单位）
- **行为**: 每 8-15 秒随机跃出水面 0.3 秒
- **特效**: 跃出时带小水花粒子（5-10 个微小水珠）
- **触发距离**: 20 米内可见

**3. 鸟类聚集（可选增强）**
- **类型**: 简化鸟类模型或精灵
- **数量**: 2-4 只
- **行为**: 在钓点上空 5-10 米盘旋
- **动画**: 简单的圆周飞行路径
- **触发距离**: 25 米内可见
- **频率**: 30% 的钓点有此特征（增加变化性）

#### 视觉层次与渐变

**距离梯度表**:

| 距离 | 水面反光 | 鱼影 | 鸟类 | 整体可见度 |
|------|----------|------|------|------------|
| 30m+ | 不可见 | 不可见 | 不可见 | 0% |
| 25-30m | 微弱闪烁 | 不可见 | 模糊轮廓 | 20% |
| 20-25m | 可见但模糊 | 偶尔可见 | 清晰 | 40% |
| 15-20m | 清晰可见 | 清晰可见 | 清晰 | 60% |
| <15m | 进入"靠近阶段" | - | - | - |

### 听觉设计（权重 30%）

#### 环境音效层

**1. 方向性水流声**
- **音效类型**: 轻柔流水声循环
- **音量**: 基础环境音 +10% 音量
- **空间化**: 启用 3D 音效，从钓点位置发出
- **频率**: 中低频 (200-800 Hz)
- **触发距离**: 20 米内可听
- **音量衰减**: 距离平方反比衰减

**2. 鸟鸣点缀（配合视觉）**
- **音效类型**: 海鸥/水鸟叫声
- **触发条件**: 仅当鸟类视觉元素存在时
- **频率**: 每 10-20 秒一次
- **音量**: 环境音 +5%
- **空间化**: 来自鸟类位置

**3. 微弱"叮"声（神秘暗示）**
- **音效类型**: 类似风铃的清脆音
- **音量**: 极低，环境音 +3%
- **触发**: 随机，每 15-30 秒
- **音调**: 高频 (1500-2000 Hz)
- **目的**: 潜意识吸引注意力

#### 音效混合策略

```
总音量 = 基础环境音 (100%) + 水流声 (10%) + 鸟鸣 (5%) + 叮声 (3%)
最大增幅 = 18%（不会压过环境音）
```

### 交互设计（权重 10%）

#### 无强制交互

**原则**: 阶段 1 应该是"被动发现"，不打断玩家的自由探索

**实现**:
- 无 UI 提示
- 无相机引导
- 无移动速度变化
- 无准星/瞄准变化

**唯一的隐性交互**: 玩家的视角方向会轻微影响水面反光的可见度（面向钓点时更明显）

## 阶段 2: 靠近阶段的氛围营造

### 视觉设计（权重 60%）

#### 渐进视觉强化

**1. 钓点标识显现**
- **类型**: 圆形光圈或波纹图案
- **初始状态** (15米):
  - 透明度: 20%
  - 颜色: 浅蓝色 (#4682B4)
  - 大小: 直径 2 米
  - 边缘模糊度: 高 (80%)
  
- **中间状态** (10米):
  - 透明度: 40%
  - 颜色: 中蓝色 (#1E90FF)
  - 大小: 直径 2.5 米
  - 边缘模糊度: 中 (50%)
  
- **靠近状态** (5米):
  - 透明度: 60%
  - 颜色: 亮蓝色 (#00BFFF)
  - 大小: 直径 3 米
  - 边缘模糊度: 低 (20%)

**2. 水纹动画增强**
- **类型**: 同心圆涟漪扩散
- **频率**: 15米时每 3 秒一圈，5米时每 1.5 秒一圈
- **扩散速度**: 0.5 米/秒
- **透明度**: 随距离衰减 (中心 60% → 边缘 0%)

**3. 颜色饱和度提升**
- **实现**: Shader 参数调整
- **公式**: `饱和度 = 基础饱和度 × (1 + 距离系数 × 0.4)`
- **距离系数**: (15m - 当前距离) / 15m
- **效果**: 越近，颜色越鲜艳

**4. 细节元素显现**
- **鱼影频率增加**: 15米时每15秒 → 5米时每8秒
- **水草摇曳** (5米内): 在钓点周围添加微动水草
- **光柱效果** (首次发现): 从钓点射向天空的微弱光束

#### 视觉渐变曲线

```
视觉强度 %
60  │                    ╱──────
    │                 ╱
40  │              ╱
    │           ╱
20  │     ──╱
    │   ╱
0   └─────────────────────────
    15m  12m  10m  8m  5m (距离)
```

### 听觉设计（权重 30%）

#### 音效层次增强

**1. 水流声增强**
- **音量变化**: 
  - 15米: +10%
  - 10米: +15%
  - 5米: +20%
- **频率变化**: 随距离增加高频成分（清脆感）
- **立体声增强**: 3D 定位更明确

**2. 环境音乐淡入** (可选)
- **触发距离**: 10 米内
- **音乐类型**: 轻柔的探索主题（弦乐+竖琴）
- **音量**: 极低，背景音乐 +5%
- **淡入时间**: 2 秒

**3. 心跳音效** (首次发现强化)
- **触发**: 仅首次发现
- **类型**: 低频鼓点
- **节奏**: 60-80 BPM（与人心跳接近）
- **音量**: 极低，潜意识感知
- **距离**: 8 米内触发

### 交互设计（权重 10%）

#### 轻微相机引导

**实现方式**:
- **触发距离**: 10 米内
- **效果**: 相机倾向钓点方向偏移 3-5 度
- **速度**: 缓慢，0.5 秒完成倾斜
- **可取消性**: 玩家移动视角可完全覆盖
- **目的**: 暗示而非强制

#### 移动速度微调

**触发距离**: 8 米内
**效果**: 移动速度降低 10%
**原因**: 营造"放慢脚步，仔细观察"的沉浸感
**可取消性**: 玩家冲刺可覆盖

#### 准星/瞄准吸附

**触发距离**: 5 米内
**效果**: 准星靠近钓点时有微弱吸附感（偏移 2-3 度）
**强度**: 极轻，不影响正常瞄准
**目的**: 潜意识提示"这里可交互"

## 阶段 3: 确认阶段的氛围营造

### 视觉设计（权重 50%）

#### 清晰的交互提示

**1. UI 提示出现**
- **类型**: 圆形图标 + 文字提示
- **位置**: 屏幕中央偏下，或世界空间钓点上方 1 米
- **图标**: 钓鱼竿图标或 "E" 键图标
- **文字**: "按 E 开始钓鱼" （支持多语言）
- **字体**: 清晰易读，描边增强可见度
- **大小**: 中等，不遮挡视野
- **动画**: 淡入 (0.3秒) + 轻微脉冲 (1秒周期，缩放 95%-105%)

**2. 钓点标识最终状态**
- **透明度**: 100%
- **颜色**: 鲜艳蓝色 (#00BFFF) 或主题色
- **光效**: 添加外发光效果 (Bloom)
- **边缘**: 清晰锐利
- **脉冲动画**: 1 秒周期呼吸效果

**3. 首次发现特效**
- **光芒闪烁**: 0.5 秒的亮光爆发
- **粒子爆发**: 20-30 个粒子从中心向外扩散
- **"NEW!" 标签**: 红色或金色小标签
- **持续时间**: 2 秒后淡出，仅保留标准提示

### 听觉设计（权重 30%）

#### 确认音效

**1. 主确认音**
- **类型**: 清脆的"叮"声或竖琴弹奏
- **音调**: 高频 (2000-3000 Hz)
- **音量**: 明显但不刺耳，音效音量 80%
- **时长**: 0.3-0.5 秒
- **触发**: 进入 5 米范围瞬间

**2. 首次发现音效**
- **类型**: 特殊和弦 + 闪光音效
- **组成**: "叮" + "bong" (低音共鸣)
- **音量**: 音效音量 100%
- **时长**: 1 秒
- **独特性**: 与标准确认音明显不同

**3. 环境音淡化**
- **效果**: 环境音轻微降低 10%
- **目的**: 突出确认音和 UI 提示
- **恢复**: 1 秒后恢复正常

### 交互设计（权重 20%）

#### 交互状态激活

**1. 交互键可用**
- **键位**: E 键 (可自定义)
- **状态**: 从禁用变为可用
- **视觉反馈**: 键位图标高亮

**2. 准星锁定** (可选)
- **效果**: 准星变为"锁定"状态图标
- **颜色**: 从白色变为绿色/蓝色
- **目的**: 明确"已对准交互对象"

**3. 角色动作** (可选)
- **效果**: 角色停下脚步，或转向钓点
- **条件**: 仅当玩家松开移动键时
- **时长**: 0.5 秒

**4. 触觉反馈** (控制器)
- **类型**: 轻微震动
- **强度**: 20%
- **时长**: 0.2 秒
- **触发**: 进入交互范围瞬间

## 仪式设计：首次发现的完整流程

### 仪式结构

```
准备 → 触发 → 展开 → 高潮 → 余韵
```

### 仪式时间线

```
0s        2s       5s       8s      10s
│─准备──│─触发──│──展开──│─高潮─│─余韵─│
远处看到  靠近    钓点清晰  确认   图鉴提示
```

### 详细步骤

#### 1. 准备 (0-2秒)

**玩家看到**:
- 远处的微光和鸟类

**玩家感受**:
- "那是什么？"

**系统行为**:
- 激活钓点视觉元素
- 开始播放环境音

#### 2. 触发 (2-5秒)

**玩家行为**:
- 决定靠近

**玩家感受**:
- "去看看吧"

**系统行为**:
- 开始渐变视觉强度
- 增强音效
- 可能触发探索音乐

#### 3. 展开 (5-8秒)

**玩家看到**:
- 钓点标识逐渐清晰
- 水纹、光效增强

**玩家感受**:
- "越来越清楚了，期待！"

**系统行为**:
- 视觉/听觉持续强化
- 相机轻微引导
- 移动速度微调

#### 4. 高潮 (8-9秒)

**玩家看到**:
- UI 提示出现
- 光芒爆发（首次）
- "NEW!" 标签

**玩家听到**:
- 清脆的确认音
- 特殊和弦（首次）

**玩家感受**:
- "就是这里！"
- "我发现了新钓点！"（首次）

**系统行为**:
- 激活交互功能
- 播放确认音效
- 触觉反馈

#### 5. 余韵 (9-10秒)

**玩家看到**:
- 图鉴提示"钓点已记录"

**玩家行为**:
- 按 E 开始钓鱼，或
- 离开继续探索

**系统行为**:
- 保持 UI 提示
- 等待玩家交互
- 首次发现：记录到图鉴

## 技术实现要点

### 视觉技术

**1. 距离检测系统**
- 使用球形碰撞体 (Sphere Collider) 检测玩家距离
- 三个距离阈值: 30m, 15m, 5m
- 每帧更新距离计算（优化：仅当玩家在 40m 内）

**2. 渐变插值**
- 使用 Lerp 函数实现平滑过渡
- 插值速度: 0.1-0.3 秒
- 参数: 透明度、颜色、大小、模糊度

**3. Shader 参数**
- 饱和度控制
- 外发光 (Bloom)
- 边缘模糊 (Blur)

**4. 粒子系统**
- 水面反光: Particle System, 5-8 粒子
- 水花: 子发射器，10 粒子/次
- 光粒子（首次）: 20-30 粒子，径向发射

### 听觉技术

**1. 3D 音效**
- 启用空间化 (Spatial Audio)
- 距离衰减: 平方反比
- 最大听距: 30 米

**2. 音量混合**
- 使用 Audio Mixer 控制各层音量
- 动态调整环境音/音效比例

**3. 音效 CD 时间**
- 确认音: 5 秒 CD（同一钓点）
- 防止玩家反复进出触发

### 交互技术

**1. 相机控制**
- 使用 Cinemachine 或 Camera Slerp
- 目标偏移: 钓点方向 3-5 度
- 平滑时间: 0.5 秒
- 玩家输入优先级更高

**2. 移动速度调整**
- 修改玩家 Character Controller 的速度参数
- 渐变应用，避免突变

**3. 准星吸附**
- 射线检测 (Raycast) 目标
- 偏移角度: 2-3 度
- 吸附力: 极轻

### 性能优化

**1. LOD 系统**
- 远距离 (20m+): 低细节（仅反光）
- 中距离 (10-20m): 中细节
- 近距离 (<10m): 全细节

**2. 对象池**
- 粒子系统复用
- 音效 Audio Source 复用

**3. 更新频率**
- 距离检测: 每 0.1 秒（非每帧）
- 视觉更新: 每帧
- 音效更新: 每 0.2 秒

## 验收标准

### 视觉验收

- [ ] 三阶段视觉强度渐变平滑，无跳变
- [ ] 首次/重复发现视觉差异明显
- [ ] 距离 30 米外不可见，5 米内完全清晰
- [ ] 颜色、透明度、大小的渐变符合设计曲线
- [ ] 多个钓点同时在场时，视觉互不干扰

### 听觉验收

- [ ] 3D 音效方向性正确
- [ ] 音量渐变符合距离衰减规律
- [ ] 确认音清晰可辨，不刺耳
- [ ] 首次发现音效独特，令人印象深刻
- [ ] 多钓点音效不会混杂成噪音

### 交互验收

- [ ] 相机引导轻微，不干扰玩家控制
- [ ] 移动速度调整自然，玩家能感知但不突兀
- [ ] 准星吸附极轻，不影响正常操作
- [ ] UI 提示清晰易读，位置合理
- [ ] 触觉反馈（如有）强度适中

### 仪式验收（首次发现）

- [ ] 完整仪式流程 10 秒左右
- [ ] 五阶段（准备→触发→展开→高潮→余韵）清晰可辨
- [ ] 高潮时刻有明显的"啊哈"感
- [ ] 图鉴记录提示清晰
- [ ] 玩家感到"我发现了新东西"的满足感

### 边界情况验收

- [ ] 快速跑过不会卡顿或报错
- [ ] 徘徊时视觉/听觉跟随距离动态变化
- [ ] 多钓点场景下，每个钓点独立运作
- [ ] 远离后再靠近，30 秒阈值生效
