# IDLE-F-02: "钓点吸引力"感受定义

> **任务ID**: IDLE-F-02  
> **任务名称**: 定义钓点"吸引力"的感受属性  
> **优先级**: P0  
> **当前阶段**: Phase 1 - 感受定义  
> **创建时间**: 2026-01-13

---

## 核心感受目标

### 吸引力的本质

**核心问题**: "什么让玩家想要靠近这个钓点？"

**吸引力 ≠ 发现**:
- IDLE-F-01 解决了"如何发现"（玩家意识到钓点的存在）
- IDLE-F-02 解决了"为什么被吸引"（玩家产生靠近的动机）
- 吸引力是发现的前提和驱动力

**设计理念**:  
吸引力不是单一的"强度数值"，而是视觉、空间、暗示三个维度的复合感受。不同玩家对不同维度的敏感度不同，但综合吸引力决定了他们是否会主动探索。

## 维度 1: 视觉吸引力

### 感受描述

**玩家此刻的心理状态**:
- "那边有什么在发光？"（好奇）
- "颜色看起来不一样"（注意到异常）
- "有东西在动"（动态吸引注意力）

**感受属性**:
| 属性 | 值 | 说明 |
|------|-----|------|
| **感受强度** | 分级 (20-100%) | 根据视觉元素的组合决定 |
| **持续时间** | 持续 | 只要在视野内就持续存在 |
| **情绪色彩** | 好奇、兴奋 | 取决于视觉的独特性 |
| **注意力需求** | 低-高 | 微弱需要主动观察，强烈自动吸引 |

### 强度分级

#### 微弱视觉吸引 (20-30%)

**适用**: 普通钓点

**视觉元素**:
- **颜色**: 水面微微偏蓝（色温差异 5-10%）
- **光效**: 2-3 个微小光点，透明度 30%，缓慢闪烁（1.5 秒周期）
- **动态**: 1 个鱼影每 15 秒跃出一次
- **对比度**: 与环境差异小（+10% 饱和度）

**玩家感受**: "那边好像有点不一样，但不确定"

**设计意图**: 需要玩家主动观察和探索，而非被动接收

#### 明显视觉吸引 (50-70%)

**适用**: 稀有钓点

**视觉元素**:
- **颜色**: 水面明显偏蓝绿色（色温差异 20-30%）
- **光效**: 5-8 个光点，透明度 60%，有节奏闪烁（1 秒周期）
- **动态**: 2-3 个鱼影每 10 秒跃出，鸟类 3-5 只在上空盘旋
- **对比度**: 与环境差异明显（+30% 饱和度）
- **额外元素**: 水面涟漪持续扩散

**玩家感受**: "那里肯定有什么，得去看看"

**设计意图**: 在正常游戏过程中能被注意到，但不强制

#### 强烈视觉吸引 (80-100%)

**适用**: 传说钓点

**视觉元素**:
- **颜色**: 独特的金色/银色光芒（完全不同的色系）
- **光效**: 光柱垂直射向天空，15-20 个粒子环绕旋转
- **动态**: 多条鱼跃出，水面翻涌，鸟类聚集并俯冲
- **对比度**: 极高对比（+60% 饱和度，+40% 亮度）
- **额外元素**: 特殊粒子效果，光环脉冲，雾气缭绕

**玩家感受**: "那是什么？！必须去看！"

**设计意图**: 几乎不可能错过，创造"特殊时刻"

### Socratic Questioning

**Q1: 为什么视觉吸引力占 40% 权重？**  
A: 因为视觉是最直接的感知通道。人类对视觉异常（颜色、动态、光）的反应最快。40% 确保视觉是主要但非唯一的吸引力来源。

**Q2: 微弱吸引力 20-30% 会不会太低？**  
A: 这是有意为之。普通钓点不应该太显眼，否则失去探索的乐趣。玩家学会"读水"后能够轻松识别这些微弱信号，这本身就是技能成长。

**Q3: 颜色差异为什么用"色温"而非"颜色"？**  
A: 色温差异更自然。环境中的水面本来就有色温变化（阳光、阴影），色温偏移更符合自然规律，而突兀的颜色（如纯红色水面）会破坏真实感。

**Q4: 传说钓点的光柱会不会太"游戏化"？**  
A: 传说钓点本身就是"特殊事件"，适度的游戏化是可接受的。而且光柱可以叙事化为"月神的祝福"等，不完全是UI元素。

## 维度 2: 空间吸引力

### 感受描述

**玩家此刻的心理状态**:
- "那个地方看起来很特别"（地形吸引）
- "拐角后面有什么？"（神秘感）
- "那个位置感觉不错"（空间直觉）

**感受属性**:
| 属性 | 值 | 说明 |
|------|-----|------|
| **感受强度** | 分级 (20-100%) | 根据空间位置和地形决定 |
| **持续时间** | 渐变 | 随玩家移动而变化 |
| **情绪色彩** | 好奇、期待 | 空间的神秘性决定 |
| **注意力需求** | 中 | 需要空间感知能力 |

### 强度分级

#### 微弱空间吸引 (20-30%)

**适用**: 普通钓点

**空间特征**:
- **视野位置**: 视野边缘（60-80 度偏角）
- **地形引导**: 无明显引导，随机位置
- **可见性**: 完全可见，无遮挡
- **周围环境**: 普通环境，无特殊地标

**玩家感受**: "那里也能钓鱼，但不特别"

**设计意图**: 填充世界，提供功能，但不刻意引导

#### 明显空间吸引 (50-70%)

**适用**: 稀有钓点

**空间特征**:
- **视野位置**: 视野中心偏侧（20-40 度偏角）
- **地形引导**: 河流交汇、山谷尽头、湖心小岛
- **可见性**: 部分遮挡（树木、岩石）增加神秘感
- **周围环境**: 特殊地形（瀑布、古树、奇石）

**玩家感受**: "那个位置看起来值得探索"

**设计意图**: 地形自然引导玩家探索

#### 强烈空间吸引 (80-100%)

**适用**: 传说钓点

**空间特征**:
- **视野位置**: 视野正中央（0-10 度偏角）或"视野之外但路径明确"
- **地形引导**: 多重地形引导（道路、河流、山脉）汇聚于此
- **可见性**: 若隐若现（雾气、光影、距离）最大化神秘感
- **周围环境**: 独特地标（隐藏山谷、月光湖、古老祭坛）

**玩家感受**: "那里一定有重要的东西"

**设计意图**: 创造"朝圣"般的探索体验

### Socratic Questioning

**Q1: 为什么空间吸引力占 30% 权重？**  
A: 空间感知是更高级的认知能力，不是所有玩家都敏感。30% 确保空间设计有价值，但不依赖于它。视觉吸引力更普适。

**Q2: 部分遮挡为什么增加吸引力？**  
A: 心理学的"不完全信息"效应。完全可见的东西不神秘，完全不可见的东西不存在。若隐若现最能激发好奇心。

**Q3: 视野正中央和视野之外哪个更吸引人？**  
A: 视野正中央是"立即吸引"，视野之外是"探索吸引"。前者适合近距离发现，后者适合远距离（如山顶看到远处的湖泊）。

## 维度 3: 暗示吸引力

### 感受描述

**玩家此刻的心理状态**:
- "鸟在那里聚集，肯定有鱼"（推理）
- "听起来水声很丰富"（听觉暗示）
- "这里的环境像是会有好钓点"（经验判断）

**感受属性**:
| 属性 | 值 | 说明 |
|------|-----|------|
| **感受强度** | 分级 (20-100%) | 根据暗示的数量和质量决定 |
| **持续时间** | 持续 | 暗示持续存在 |
| **情绪色彩** | 推理、满足 | "我猜对了"的感觉 |
| **注意力需求** | 高 | 需要观察和学习 |

### 强度分级

#### 微弱暗示吸引 (20-30%)

**适用**: 普通钓点

**暗示元素**:
- **生态暗示**: 1-2 只鸟在水面上方（不聚集）
- **听觉暗示**: 标准水流声（+5% 音量）
- **视觉线索**: 无人工痕迹
- **神秘元素**: 无

**玩家感受**: "这里可能有鱼，试试看"

**设计意图**: 提供基础线索，但不明显

#### 明显暗示吸引 (50-70%)

**适用**: 稀有钓点

**暗示元素**:
- **生态暗示**: 3-5 只鸟聚集，偶尔俯冲抓鱼，水面昆虫飞舞
- **听觉暗示**: 丰富水流声（+15% 音量），鸟鸣声，水花声
- **视觉线索**: 岸边有旧渔具痕迹（磨损的石头、废弃的鱼篓）
- **神秘元素**: 轻微的雾气或光影变化

**玩家感受**: "这些迹象表明这是个好钓点"

**设计意图**: 多重线索相互印证，让玩家有"侦探"的满足感

#### 强烈暗示吸引 (80-100%)

**适用**: 传说钓点

**暗示元素**:
- **生态暗示**: 大量鸟类（10+），鱼群频繁跃出，水面生物活跃
- **听觉暗示**: 特殊音乐主题淡入，神秘钟声，水声如歌
- **视觉线索**: 古老的石碑、渔夫的传说故事标记
- **神秘元素**: 强烈的神秘氛围（月光、萤火虫、奇异植物发光）

**玩家感受**: "这里一定有传说中的东西"

**设计意图**: 创造"发现宝藏"的史诗感

### Socratic Questioning

**Q1: 为什么暗示吸引力也占 30%？**  
A: 暗示是"学习型"吸引力。新手可能忽略，但老手能快速识别。30% 确保学习有价值，熟练玩家能从暗示获得显著优势。

**Q2: 旧渔具痕迹会不会破坏"原始自然"的氛围？**  
A: 不会，反而增强真实感。真实世界的好钓点都是世代渔夫传承的，人类痕迹是历史的一部分，而非破坏。

**Q3: 音乐主题淡入算"暗示"还是"氛围"？**  
A: 两者兼有。在暗示维度，它"告诉"玩家这里特殊。在氛围维度（Phase 2），它营造感觉。这种重叠是设计的整体性。

## 综合吸引力公式

### 计算方法

```
基础吸引力 = 视觉吸引力 × 0.4 + 空间吸引力 × 0.3 + 暗示吸引力 × 0.3
```

### 案例计算

#### 案例 1: 普通钓点

```
视觉: 25% (微光 + 少量鱼影)
空间: 20% (视野边缘，无地形引导)
暗示: 25% (1-2 只鸟)

综合吸引力 = 0.25×0.4 + 0.20×0.3 + 0.25×0.3 = 0.235 = 23.5%
```

**玩家体验**: 需要主动观察才会注意到

#### 案例 2: 稀有钓点

```
视觉: 60% (明显光效 + 鸟类聚集)
空间: 70% (河流交汇，视野中心偏侧)
暗示: 65% (多重环境暗示)

综合吸引力 = 0.60×0.4 + 0.70×0.3 + 0.65×0.3 = 0.645 = 64.5%
```

**玩家体验**: 正常游戏会注意到并产生探索欲望

#### 案例 3: 传说钓点

```
视觉: 95% (光柱 + 特效)
空间: 100% (独特地点，视野正中或强引导)
暗示: 90% (神秘音乐 + 生态异常)

综合吸引力 = 0.95×0.4 + 1.00×0.3 + 0.90×0.3 = 0.95 = 95%
```

**玩家体验**: 几乎不可能错过，立即产生探索冲动

## 距离衰减模型

### 衰减公式

```
有效吸引力 = 基础吸引力 × 衰减系数

衰减系数 = (1 - distance / max_distance)²
```

### 衰减表

#### 普通钓点 (max_distance = 20m)

| 距离 | 基础吸引力 23.5% | 衰减系数 | 有效吸引力 |
|------|-----------------|----------|------------|
| 20m | 23.5% | 0% | 0% |
| 15m | 23.5% | 6.25% | 1.5% |
| 10m | 23.5% | 25% | 5.9% |
| 5m | 23.5% | 56.25% | 13.2% |
| 2m | 23.5% | 81% | 19% |

**解读**: 普通钓点需要非常近（5m内）才有明显吸引力

#### 稀有钓点 (max_distance = 30m)

| 距离 | 基础吸引力 64.5% | 衰减系数 | 有效吸引力 |
|------|-----------------|----------|------------|
| 30m | 64.5% | 0% | 0% |
| 25m | 64.5% | 2.8% | 1.8% |
| 20m | 64.5% | 11.1% | 7.2% |
| 15m | 64.5% | 25% | 16.1% |
| 10m | 64.5% | 44.4% | 28.6% |
| 5m | 64.5% | 69.4% | 44.8% |

**解读**: 稀有钓点在 10-15m 开始有吸引力，5m 时非常明显

#### 传说钓点 (max_distance = 50m)

| 距离 | 基础吸引力 95% | 衰减系数 | 有效吸引力 |
|------|---------------|----------|------------|
| 50m | 95% | 0% | 0% |
| 40m | 95% | 4% | 3.8% |
| 30m | 95% | 16% | 15.2% |
| 20m | 95% | 36% | 34.2% |
| 10m | 95% | 64% | 60.8% |
| 5m | 95% | 81% | 77% |

**解读**: 传说钓点在 30m 就有吸引力，20m 时非常明显

## 边界与极端情况

### 边界情况 1: 三维度不一致

**场景**: 视觉强但空间弱、暗示弱

**例子**: 光效明显（80%）但位于视野边缘（20%），无环境暗示（15%）

**综合吸引力**: 0.80×0.4 + 0.20×0.3 + 0.15×0.3 = 0.425 = 42.5%

**设计应对**:
- 这是合理的组合，代表"刻意放置的视觉标记"
- 适用于引导玩家到特定位置
- 但缺少自然感，应谨慎使用

### 边界情况 2: 玩家对吸引力的抗性

**场景**: 玩家在执行任务，忽略钓点吸引力

**感受影响**:
- 即使高吸引力（80%）也可能被忽略
- 玩家的注意力被任务目标占据

**设计应对**:
- 这是正常的，不强制吸引
- 可以用音效作为"备用提示"
- 任务完成后玩家自然回到探索模式

### 边界情况 3: 吸引力疲劳

**场景**: 连续遇到多个高吸引力钓点

**感受影响**:
- 第一个: "哇，稀有钓点！"
- 第二个: "又一个，不错"
- 第三个: "好多啊...麻木了"

**设计应对**:
- 稀有钓点密度控制（最小间距 200m）
- 传说钓点极度稀少（全地图 1-3 个）
- 用时间/条件限制（特定时间才出现）

### 边界情况 4: 吸引力与发现的时间差

**场景**: 玩家被吸引但未"发现"（进入 IDLE-F-01 流程）

**时间线**:
1. 远距离（25m）: 感受到吸引力
2. 靠近（20m → 10m）: 吸引力增强
3. 接近（5m）: 触发"发现"流程

**设计应对**:
- 吸引力是"拉力"，发现是"确认"
- 两者平滑衔接，无跳变
- 吸引力的"好奇"自然过渡到发现的"好奇"

## 差异化设计：普通 vs 稀有 vs 传说

### 综合对比表

| 维度 | 普通钓点 | 稀有钓点 | 传说钓点 |
|------|----------|----------|----------|
| **视觉吸引** | 20-30% | 50-70% | 80-100% |
| **空间吸引** | 20-30% | 50-70% | 80-100% |
| **暗示吸引** | 20-30% | 50-70% | 80-100% |
| **综合吸引力** | 20-30% | 50-70% | 80-100% |
| **最大距离** | 20m | 30m | 50m |
| **有效距离** | <10m | 10-20m | 20-40m |
| **发现频率** | 高（每分钟） | 中（每5-10分钟） | 低（每小时或特殊条件） |
| **错过容忍度** | 高（可以错过） | 中（建议不错过） | 低（几乎不会错过） |

### 玩家学习曲线

**新手阶段**:
- 主要依赖视觉吸引力（显眼的光效和鱼影）
- 可能错过暗示微弱的普通钓点
- 传说钓点的强烈吸引确保不会错过

**熟练阶段**:
- 开始注意空间吸引力（地形特征）
- 学会识别生态暗示（鸟类、水声）
- 能快速定位普通钓点

**专家阶段**:
- 仅凭暗示就能判断钓点质量
- 主动探索有空间吸引力的位置
- "一眼就知道"哪里有好钓点

## 验收标准

### 感受层面

- [ ] 玩家能感受到三个维度的吸引力差异
- [ ] 不同强度的吸引力产生不同的探索动机
- [ ] 普通/稀有/传说钓点的吸引力差异明显
- [ ] 玩家有"学习识别"的成长感

### 数值层面

- [ ] 吸引力公式合理，三维度权重平衡
- [ ] 距离衰减符合感知规律
- [ ] 边界情况处理合理

### 整体体验

- [ ] 吸引力与发现流程（IDLE-F-01）平滑衔接
- [ ] 吸引力不过度引导，保持探索自由
- [ ] 学习曲线合理，新手到专家都有适配
