## 1. 感受定义 (3 任务)

### HOOK-F-01: 定义咬钩瞬间的"惊喜爆发"
- **优先级**: P0
- **设计产出**: 从平静到爆发的感受设计

### HOOK-F-02: 定义2秒窗口的"紧张感"
- **优先级**: P0
- **设计产出**: 时间压力的感受强度设计

### HOOK-F-03: 定义不同鱼类咬钩的力度差异
- **优先级**: P1
- **设计产出**: 小鱼/大鱼的咬钩感受对比

## 2. 氛围营造 (5 任务)

### HOOK-A-01: 设计咬钩的视觉爆发特效
- **优先级**: P0
- **设计产出**: 鱼漂下沉+水花+光效的组合

### HOOK-A-02: 设计咬钩的音效分层
- **优先级**: P0
- **设计产出**: 提示音+水声+鱼竿震动音

### HOOK-A-03: 设计屏幕的震动/闪烁效果
- **优先级**: P1
- **设计产出**: 轻微震动、边缘高光的设计

### HOOK-A-04: 设计咬钩时的慢动作效果
- **优先级**: P2
- **设计产出**: 0.5秒慢动作的可选设计

### HOOK-A-05: 设计不同稀有度的咬钩差异
- **优先级**: P1
- **设计产出**: 普通/稀有鱼的咬钩视觉差异

## 3. 仪式设计 (4 任务)

### HOOK-R-01: 设计UI提示的爆发式出现
- **优先级**: P0
- **设计产出**: "收杆！"文字的动画设计

### HOOK-R-02: 设计收杆按键的反馈
- **优先级**: P0
- **设计产出**: 按下瞬间的视觉/触觉反馈

### HOOK-R-03: 设计首次咬钩的教学强化
- **优先级**: P1
- **设计产出**: 慢动作+标注+额外时间

### HOOK-R-04: 设计咬钩倒计时的隐性提示
- **优先级**: P1
- **设计产出**: 进度圈、闪烁加速等设计

## 4. 故事包裹 (2 任务)

### HOOK-S-01: 设计角色的惊讶反应
- **优先级**: P2
- **设计产出**: 表情、动作、语音的设计

### HOOK-S-02: 设计稀有鱼咬钩的特殊暗示
- **优先级**: P1
- **设计产出**: 独特音效、光效暗示稀有度

## 5. 无瑕打磨 (2 任务)

### HOOK-P-01: 检查2秒窗口的精确性
- **优先级**: P0
- **设计产出**: 网络延迟、帧率波动的补偿

### HOOK-P-02: 检查误操作的容错处理
- **优先级**: P1
- **设计产出**: 提前按键、连续按键的处理
