# Workflow Authoring 前置检查清单

> 在开始编写工作流之前，检查以下事项

## 需求确认

- [ ] 明确工作流的触发方式（手动/事件/定时）
- [ ] 明确工作流的核心职责
- [ ] 明确预期的输出类型

## 权限规划

- [ ] 列出需要的最小权限
- [ ] 确认是否需要 safe-outputs

## 工具规划

- [ ] 确定需要使用的工具（github/bash/edit/MCP）
- [ ] 检查是否有可复用的 imports
