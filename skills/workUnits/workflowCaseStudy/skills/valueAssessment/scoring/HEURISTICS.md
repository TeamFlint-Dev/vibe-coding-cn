# 快速筛选启发式规则

## 🟢 优先选择

| 信号 | 原因 |
|------|------|
| 最近被频繁修改 | 说明在积极解决问题，有演进痕迹 |
| 文件名暗示新功能 | 可能包含最新模式 |
| 使用了多个 MCP | 工具协作模式值得学习 |
| 有 `cache-memory` | 状态管理是高级话题 |
| 触发方式复杂 | 多触发器设计有参考价值 |

## 🔴 暂时跳过

| 信号 | 原因 |
|------|------|
| 文件非常短 (<50行) | 可能太基础 |
| 已有多个类似案例 | 边际价值低 |
| 文件名带 `example/demo` | 通常是入门示例 |
| 只使用基础工具 | 学习空间有限 |

---

## 时间衰减因子

| 时间因素 | 影响 |
|---------|------|
| 刚发布的新工作流 | +20% 价值 |
| 最近有重大更新 | +15% 价值 |
| 长期未更新 | -10% 价值 |
| 被标记为 deprecated | -50% 价值 |
