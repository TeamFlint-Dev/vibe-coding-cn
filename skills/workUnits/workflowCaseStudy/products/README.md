# 产品 (Products)

这里存放实际产出的工作流及其设计文档。

---

## 目录结构

每个产品一个子目录：

```
products/
├── {workflow-name}/
│   ├── design.md          # 设计文档（为什么这样设计）
│   ├── {workflow-name}.md # 工作流源文件
│   └── changelog.md       # 变更记录（可选）
```

---

## 产品 vs 资产

| 类型 | 说明 | 位置 |
|------|------|------|
| **产品** | 可直接使用的工作流 | `products/` |
| **资产** | 调研报告、模式提炼 | `reports/`, `skills/` |

产品是最终交付物。资产是生产产品的原材料和知识积累。
