# 美术技能

此目录用于存放与游戏美术相关的技能，包括但不限于：

- 2D 美术资产制作
- 3D 建模与材质
- 动画与特效
- UI 视觉设计
- 风格指南与规范

## 技能结构

待添加实际技能时，应遵循以下结构：

```
art/
├── SKILL.md          # 技能总述
├── shared/           # 共享资源
│   ├── references/   # 参考文档
│   └── templates/    # 模板文件
└── [subSkill]/       # 子技能目录
    └── SKILL.md
```
