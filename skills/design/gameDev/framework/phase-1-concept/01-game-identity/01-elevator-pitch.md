# 01 电梯演讲 (Elevator Pitch)

> 用30秒让别人理解并想玩这个游戏

## 🎯 核心问题

**如果你在电梯里遇到一个玩家/投资人/合作伙伴，如何在30秒内让他对你的游戏感兴趣？**

## 📋 任务说明

电梯演讲是一个30秒的口头介绍，目标是让听众：
1. **理解**游戏是什么
2. **记住**游戏的核心特点
3. **产生兴趣**想要了解更多或试玩

一个好的电梯演讲应该包含：
- **What**：游戏是什么类型
- **Who**：给谁玩的
- **Why**：为什么有趣/独特
- **Hook**：最吸引人的点

## 🔍 引导问题

1. 如果只能说一句话，这个游戏最核心的是什么？
2. 这个游戏和玩家熟悉的什么游戏相似？
3. 最让你兴奋的设计点是什么？
4. 玩家为什么会选择玩这个游戏？
5. 如何在3秒内吸引注意力？

## 📝 输出模板

```
[Hook - 吸引注意力]
[用一句话说明游戏是什么]
[核心玩法/循环]
[独特价值/差异化]
[目标玩家]
```

### 示例结构

```
想象一下，[某个熟悉场景]。
这是一个 [类型] 游戏，[核心玩法]。
玩家可以 [主要行为1]、[主要行为2]、[主要行为3]。
不同于其他游戏的是，[独特点]。
适合 [目标玩家] 在 [使用场景] 游玩。
```

## ✅ 验收标准

- [ ] 总长度在30秒内（约60-80个字）
- [ ] 包含游戏类型
- [ ] 包含核心玩法
- [ ] 包含独特价值
- [ ] 听众能理解游戏是什么
- [ ] 听众产生"想试试"的想法
- [ ] 不使用行业黑话（除非面向专业听众）
- [ ] 避免空洞形容词（"有趣"、"好玩"等）

## 💡 示例

### 案例：钓鱼放置养成游戏

**电梯演讲（中文版）**：

```
你有没有想过，如果钓鱼可以全自动化会怎样？

这是一个结合放置和养成的钓鱼游戏。
你在海底建造自动钓鱼设施，收集数百种稀有鱼类，
逐步扩张你的海洋渔业帝国。

不同于传统钓鱼游戏的是，你不需要一直盯着屏幕——
鱼会自动钓起来，即使你离线也在增长。
你只需要在空闲时优化策略、解锁新区域、收集图鉴。

适合喜欢收集和成长，但没时间hardcore游戏的休闲玩家。
```

**字数统计**：约120字，朗读约25-30秒

**分析**：
- ✅ Hook：开头提出一个有趣的问题
- ✅ 类型：放置 + 养成 + 钓鱼
- ✅ 核心玩法：建造设施、收集鱼类、扩张帝国
- ✅ 独特点：自动化、离线增长
- ✅ 目标玩家：休闲玩家

### 案例：其他游戏参考

**《吸血鬼幸存者》**：
```
想象一下，《魂斗罗》遇上《暗黑破坏神》。
这是一个单摇杆射击游戏，你用一个摇杆移动，
武器自动攻击成千上万的敌人。
游戏的乐趣在于选择和组合不同的武器和升级，
创造出毁天灭地的Build，一个人对抗整个军团。
每局15-30分钟，适合碎片时间的刷刷刷体验。
```

**《星露谷物语》**：
```
如果你厌倦了城市生活，想要田园诗般的慢生活呢？
这是一个农场模拟游戏，你继承了爷爷的旧农场，
种植作物、养殖动物、结交邻居、探索洞穴。
不同于传统农场游戏，这里有RPG式的角色成长、
社交系统和剧情，让你真正沉浸在这个温暖的小镇里。
适合想要放松和逃离现实的玩家。
```

## 💭 常见错误

### ❌ 错误示例1：过于抽象

```
这是一个创新的游戏体验，融合了多种玩法元素，
为玩家提供前所未有的沉浸式娱乐。
通过独特的机制设计，玩家将体验到极致的游戏乐趣。
```

**问题**：全是空话，听众完全不知道游戏是什么。

### ❌ 错误示例2：细节过多

```
这是一个MMORPG，有15个职业，200个技能，
50个副本，3个阵营，PvP天梯赛制，公会战系统，
自由交易市场，宠物系统，坐骑系统，成就系统...
```

**问题**：信息过载，听众记不住任何东西。

### ❌ 错误示例3：没有Hook

```
这是一个钓鱼游戏。玩家可以钓鱼、收集鱼类、
升级装备。游戏有很多鱼可以钓。
```

**问题**：平淡无奇，听众提不起兴趣。

### ✅ 改进版本

```
如果钓鱼可以像《动物森友会》一样治愈，
同时又有《宝可梦》一样的收集乐趣呢？

这是一个放置钓鱼游戏，你建造自动化钓鱼设施，
收集数百种奇幻海洋生物，打造你的水下乐园。

不同于传统钓鱼游戏，你不需要等待——
离线也在成长，上线只需几分钟就能获得满足感。

适合喜欢收集和成长，但时间有限的休闲玩家。
```

## 📚 延伸阅读
