# 06 核心承诺

> 明确对玩家的核心承诺

## 🎯 核心问题

**你向玩家承诺的核心体验是什么？**

## 📋 任务说明

（此任务说明待补充，请根据具体需求填写）

本任务的目标是帮助你深入思考游戏设计中的"核心承诺"维度，确保设计的完整性和一致性。

## 🔍 引导问题

1. 从玩家角度思考：玩家会如何理解这个维度？
2. 从设计角度思考：这个维度如何影响其他设计决策？
3. 从竞品角度思考：同类游戏是如何处理这个维度的？
4. 从独特性角度思考：我们能在这个维度上做出什么创新？
