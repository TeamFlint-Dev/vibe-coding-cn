# UI/UX 设计技能

此目录用于存放与界面和用户体验设计相关的技能，包括但不限于：

- 游戏 UI 布局设计
- 交互流程设计
- HUD 设计
- 菜单系统设计
- 用户体验优化

## 技能结构

待添加实际技能时，应遵循以下结构：

```
uiUx/
├── SKILL.md          # 技能总述
├── shared/           # 共享资源
│   ├── references/   # 参考文档
│   └── templates/    # 模板文件
└── [subSkill]/       # 子技能目录
    └── SKILL.md
```
