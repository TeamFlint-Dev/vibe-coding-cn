## 根因类别说明

| 类别 | 说明 | 常见表现 |
|------|------|----------|
| 编译错误 | Verse 代码无法通过编译 | 语法错误、类型不匹配 |
| 运行时错误 | 代码编译通过但运行失败 | 空指针、逻辑错误 |
| 架构问题 | SceneGraph 架构设计缺陷 | Entity 层级混乱、事件死循环 |
| 配置错误 | UEFN 项目配置问题 | 权限不足、资源缺失 |
| 依赖问题 | 模块依赖关系错误 | 循环依赖、版本冲突 |
| 性能问题 | 代码性能不达标 | 帧率下降、内存泄漏 |
| 限制触碰 | 触碰 Verse/UEFN 的边界限制 | API 不支持、功能受限 |
