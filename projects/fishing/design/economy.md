## 二、产出设计

### 2.1 钓鱼产出（战利品）

**产出速率分析**：

| 参数 | 值 |
|------|-----|
| 平均钓鱼周期 | 10 秒（投竿 0.5s + 等待 7s + 咬钩 2s + 结算 0.5s） |
| 成功率 | 85%（考虑漏收杆） |
| 有效产出 | 0.085 鱼/秒 = 5.1 鱼/分钟 |

**稀有度产出概率**（基础，无加成）：

| 稀有度 | 概率 | 每 100 次 |
|--------|------|-----------|
| Common | 69.6% | 70 只 |
| Uncommon | 19.0% | 19 只 |
| Rare | 7.6% | 8 只 |
| Epic | 3.8% | 4 只 |

### 2.2 展示收益产出（金币）

**收益计算公式**：
```
每秒收益 = Σ(Fish.BaseIncome × RarityMultiplier)

RarityMultiplier:
  Common    = 1.0
  Uncommon  = 2.0
  Rare      = 5.0
  Epic      = 15.0
```

**收益场景模拟**：

| 场景 | 展示配置 | 每秒收益 | 每小时收益 |
|------|----------|----------|-----------|
| 开局 | 1x Common | 1 | 3,600 |
| 早期 | 3x Common | 3 | 10,800 |
| 中期 | 1x Rare + 2x Uncommon | 29 | 104,400 |
| 后期 | 1x Epic + 2x Rare | 65 | 234,000 |
| 理想 | 3x Epic | 135 | 486,000 |

### 2.3 首次发现奖励

| 稀有度 | 奖励 | 全图鉴奖励 |
|--------|------|-----------|
| Common (×2) | 10 × 2 = 20 | - |
| Uncommon (×1) | 25 | - |
| Rare (×1) | 100 | - |
| Epic (×1) | 500 | - |
| **总计** | 645 | +500 (全发现) |

## 五、产消比验证

### 5.1 短期平衡（单次游玩 30 分钟）

```
产出:
  - 展示收益（假设平均 15/秒）：15 × 1800 = 27,000 金币
  - 首发奖励（4 种鱼）：约 135 金币
  - 总产出：约 27,135 金币

消耗:
  - 升级到 Lv.6：1,318 金币

净余额：约 25,800 金币

结论：产出远大于消耗，玩家有充足升级资金 ✅
```

### 5.2 长期平衡（全流程 2 小时）

```
产出:
  - 展示收益（平均 40/秒）：40 × 7200 = 288,000 金币
  - 首发奖励（全 5 种）：645 + 500 = 1,145 金币
  - 总产出：约 289,000 金币

消耗:
  - 满级升级：7,486 金币

净余额：约 281,000 金币

结论：大量富余，需要后续内容消耗（展示位扩展、装饰等）✅
```

### 5.3 通胀控制

| 风险 | 缓解措施 |
|------|----------|
| 金币积累过快 | 后续版本加入更多消耗点 |
| 升级过于容易 | MVP 阶段优先体验流畅 |
| 无目标感 | 保留金币显示，预告未来内容 |

## 六、稀有度经济影响

### 6.1 稀有度价值对比

| 稀有度 | 展示收益 | 获取难度 | 价值评级 |
|--------|----------|----------|----------|
| Common | 1/秒 | 极易 | ⭐ |
| Uncommon | 2/秒 | 易 | ⭐⭐ |
| Rare | 5/秒 | 中 | ⭐⭐⭐ |
| Epic | 15/秒 | 难 | ⭐⭐⭐⭐⭐ |

### 6.2 鱼竿升级的经济回报

```
升级 Lv.5→Lv.6 投资分析:

费用：506 金币
效果：稀有掉落 +5%

假设每小时钓鱼 360 次:
  Lv.5 (20% 加成): Rare 9.1% → 33 只/小时
  Lv.6 (25% 加成): Rare 9.5% → 34 只/小时
  
收益差异：每小时多 1 只 Rare
额外收益：5 金币/秒 × 展示时间

回本时间：约 100 秒（如果额外 Rare 立即展示）

结论：升级投资回报率高 ✅
```

## 七、数值调参建议

### 7.1 可调参数表

| 参数 | 当前值 | 调整范围 | 影响 |
|------|--------|----------|------|
| 钓鱼等待时间 | 3-15s | 2-20s | 产出速率 |
| 稀有度权重 | 见上 | ±20% | 稀有体验 |
| 基础收益 | 1/2/5/15 | ×0.5-2 | 经济速度 |
| 升级费用 | ×1.5 | ×1.3-2 | 进度节奏 |
| 首发奖励 | 10/25/100/500 | ×0.5-2 | 早期资金 |

### 7.2 快速调整场景

**如果玩家反馈「升级太快」**：
- 升级费用乘数 1.5 → 1.8
- 或减少展示收益 30%

**如果玩家反馈「钓鱼无聊」**：
- 缩短等待时间上限 15s → 10s
- 增加稀有掉落概率

**如果玩家反馈「Epic 太难钓」**：
- Epic 基础权重 30 → 40
- 或增加鱼竿加成效果
