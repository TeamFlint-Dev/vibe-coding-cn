## 一、游戏概述

### 一句话描述

一个 UEFN 放置养成游戏，玩家通过钓鱼获取战利品，将其放置在基地中展示赚取收益，用收益升级装备以钓取更稀有的鱼。

### 游戏类型

- **主类型**：放置/挂机（Idle）
- **副类型**：收集养成（Collection）
- **平台**：UEFN (Fortnite Creative)

## 二、核心循环

```text
    ┌─────────────────────────────────────────┐
    │                                         │
    ▼                                         │
┌────────┐    ┌────────┐    ┌────────┐    ┌────────┐
│  钓鱼  │───▶│ 战利品 │───▶│  展示  │───▶│  收益  │
└────────┘    └────────┘    └────────┘    └────────┘
    ▲                                         │
    │         ┌────────┐                      │
    └─────────│  升级  │◀─────────────────────┘
              └────────┘
```

### 循环分解

| 阶段 | 玩家行为 | 系统反馈 | 情感体验 |
|------|----------|----------|----------|
| **钓鱼** | 选择钓点，投竿等待 | 鱼咬钩提示，收杆动画 | 期待、惊喜 |
| **战利品** | 查看捕获的鱼 | 稀有度展示，首次发现奖励 | 成就感、收集欲 |
| **展示** | 放置鱼到展示位 | 基地视觉变化，访客反应 | 自豪、炫耀 |
| **收益** | 离线/在线自动积累 | 金币/积分增长 | 满足感 |
| **升级** | 购买新鱼竿/解锁钓点 | 解锁更稀有鱼种 | 进步感、期待 |

## 三、目标玩家

### 主要受众

- **Fortnite 休闲玩家**：喜欢在 Creative 地图放松的玩家
- **放置类爱好者**：享受挂机收益、养成进度的玩家
- **收集控**：追求全图鉴、稀有成就的玩家

### 玩家画像

| 维度 | 描述 |
|------|------|
| 年龄 | 12-25 岁 |
| 游戏时长 | 每次 15-30 分钟，每日 2-3 次 |
| 动机 | 收集成就 > 社交展示 > 数值成长 |
| 付费意愿 | 低（依赖 Fortnite 生态，暂不设付费点） |

---

## 四、独特卖点 (USP)

### 核心差异化

1. **战利品可视化展示**
   - 不只是收集图鉴，而是真正放置在 3D 基地中
   - 其他玩家可以访问参观你的展示馆

2. **UEFN 原生体验**
   - 利用 Fortnite 的社交系统和地图发现机制
   - 无需额外下载，在 Fortnite 内直接游玩

3. **轻量级放置循环**
   - 每次游玩 5 分钟即可完成一次完整循环
   - 离线收益保证进度不停

## 五、MVP 范围定义

### ✅ MVP 包含（Must Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 单一钓点 | 一个可交互的钓鱼位置 | P0 |
| 基础钓鱼循环 | 投竿 → 等待 → 咬钩 → 收杆 | P0 |
| 5 种基础鱼 | 普通/稀有各占比例 | P0 |
| 战利品背包 | 查看已捕获的鱼 | P0 |
| 3 个展示位 | 可放置战利品的基地位置 | P0 |
| 被动收益 | 展示的鱼产生金币 | P0 |
| 1 种升级 | 鱼竿升级提高稀有率 | P0 |

### ❌ MVP 不包含（Defer）

| 功能 | 推迟原因 |
|------|----------|
| 挖矿系统 | 验证核心循环后再扩展 |
| 多钓点 | 需要更多关卡设计 |
| 社交访问 | 需要额外联机逻辑 |
| 季节活动 | 需要持续运营 |
| 成就系统 | 非核心循环 |

## 六、成功指标

### MVP 验证标准

| 指标 | 目标值 | 测量方式 |
|------|--------|----------|
| 单次游玩时长 | ≥ 10 分钟 | 游戏内计时 |
| 次日留存率 | ≥ 30% | 玩家回归比例 |
| 循环完成率 | ≥ 80% | 完成一次完整钓鱼-展示-收益循环 |
| 升级意愿 | ≥ 50% | 玩家进行首次升级的比例 |

## 七、风险与假设

### 核心假设

1. **假设**：玩家愿意为「展示战利品」停留
   - **验证方式**：追踪展示位使用率

2. **假设**：放置收益能驱动回归
   - **验证方式**：测量离线后回归时间

3. **假设**：钓鱼玩法足够有趣
   - **验证方式**：追踪单次钓鱼次数

### 已知风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| UEFN 限制 | 某些机制无法实现 | 提前验证技术可行性 |
| 内容单薄 | 快速消耗感 | 鱼种稀有度梯度设计 |
| 无差异化 | 淹没在同类地图中 | 强化展示馆视觉效果 |
