### Issue #4.1: 分段公式未定义

**位置**: `game-economy-designer/SKILL.md` L75-82

**当前内容**:
```markdown
**分段公式**：
```
IF level < threshold THEN
    value = formula_1(level)
ELSE
    value = formula_2(level)
END
```
```

**建议修改为**:
```markdown
**分段公式**：
```
IF level < threshold THEN
    value = formula_1(level)  # 例: base × level
ELSE
    value = formula_2(level)  # 例: base × (level ^ 1.5)
END

示例 (threshold=10, base=100):
- level=5:  100 × 5 = 500
- level=15: 100 × (15^1.5) ≈ 5809
```
```

**修改原因**: 提供具体公式实例，确保可验算
