## 文档维护说明

### 创建新文档时

1. 使用统一的 Markdown 格式
2. 添加文档头部元信息（类型、版本、更新日期）
3. 更新本索引文件
4. 在 SKILL.md 中添加引用

### 文档命名规范

- 使用小写字母和连字符
- 文件名应反映内容（如 `display-system.md`）
- 避免使用空格和特殊字符
