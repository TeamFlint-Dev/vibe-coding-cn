# 分支节点示例

本文档展示一个完整的分支节点 Skill 示例。

## 示例：verseDev

Verse 开发的顶层分支 Skill，包含多个子 Skill。

### 目录结构

```
verseDev/
├── SKILL.md                    # 入口文件（分支节点）
├── @changelog.md               # 变更日志
├── shared/                     # 共享资源目录
│   ├── references/             # 参考资料
│   │   └── SceneGraph.md
│   ├── api-digests/            # API 摘要
│   │   ├── verse-api.md
│   │   └── fortnite-api.md
│   └── checklists/             # 检查清单
│       └── architecture.md
├── verseOrchestrator/          # 子 Skill：开发编排
│   └── SKILL.md
├── verseArchitectureSelector/  # 子 Skill：架构选择
│   └── SKILL.md
├── verseComponent/             # 子 Skill：组件开发
│   └── SKILL.md
├── verseEventFlow/             # 子 Skill：事件流
│   └── SKILL.md
├── verseHelpers/               # 子 Skill：辅助函数
│   └── SKILL.md
└── verseProjectInit/           # 子 Skill：项目初始化
    └── SKILL.md
```

### SKILL.md 完整内容

```markdown
---
name: verseDev
description: |
  Verse 游戏开发的完整技能树。
  当进行 UEFN/Fortnite Creative 游戏开发时，使用此 Skill 选择合适的子技能。
status: stable
version: 2.0.0
created: 2024-03-01
updated: 2025-01-15
tags:
  - verse
  - uefn
  - game-development
  - orchestration
---

# Verse Development

UEFN/Fortnite Creative 游戏开发的完整技能树。

## When to Use

### 适用场景

- 开始新的 Verse 项目开发
- 需要选择合适的开发技能
- 不确定使用哪个具体子 Skill

### 作为入口

此 Skill 是 Verse 开发的入口点，帮助你定位到正确的子 Skill。

## Context Entry

### 提供

| 入口 | 类型 | 说明 |
|------|------|------|
| [shared/references/](shared/references/) | 目录 | 参考资料集合 |
| [shared/api-digests/](shared/api-digests/) | 目录 | API 摘要 |
| [shared/checklists/](shared/checklists/) | 目录 | 检查清单 |

### 依赖

| 依赖 | 类型 | 说明 |
|------|------|------|
| UEFN 环境 | 工具 | 需要安装 UEFN |
| Verse 语言基础 | 知识 | 需要了解 Verse 语法 |

## Children

### 选择指南

根据你的开发阶段和需求选择：

| 阶段 | 子 Skill | 说明 |
|------|----------|------|
| 项目启动 | [verseProjectInit](verseProjectInit/SKILL.md) | 创建新项目结构 |
| 架构设计 | [verseArchitectureSelector](verseArchitectureSelector/SKILL.md) | 选择合适架构 |
| 开发编排 | [verseOrchestrator](verseOrchestrator/SKILL.md) | 编排开发流程 |
| 组件开发 | [verseComponent](verseComponent/SKILL.md) | 开发游戏组件 |
| 事件系统 | [verseEventFlow](verseEventFlow/SKILL.md) | 设计事件流 |
| 工具函数 | [verseHelpers](verseHelpers/SKILL.md) | 通用辅助函数 |

### 决策树

```

开始 Verse 开发
    │
    ├─ 新项目？
    │   └─ 是 → verseProjectInit
    │
    ├─ 需要选择架构？
    │   └─ 是 → verseArchitectureSelector
    │
    ├─ 需要编排多个开发任务？
    │   └─ 是 → verseOrchestrator
    │
    ├─ 开发具体功能？
    │   ├─ 游戏组件 → verseComponent
    │   ├─ 事件系统 → verseEventFlow
    │   └─ 工具函数 → verseHelpers
    │
    └─ 查阅参考资料？
        └─ 查看 shared/ 目录

```

### 子 Skill 列表

#### verseProjectInit
初始化新 Verse 项目，创建标准目录结构。

#### verseArchitectureSelector
分析需求并选择合适的架构模式（SceneGraph、ECS 等）。

#### verseOrchestrator
编排复杂的开发流程，协调多个子 Skill。

#### verseComponent
开发游戏组件，包括设备、角色、UI 等。

#### verseEventFlow
设计和实现事件驱动系统。

#### verseHelpers
提供常用辅助函数和代码片段。

- 高级用户：verseOrchestrator、verseEventFlow

### 待改进
```

## 分支节点检查清单

使用此清单确保分支节点 Skill 完整：

```markdown
## 必须项

- [ ] SKILL.md 存在
- [ ] name 和 description 完整
- [ ] When to Use 说明入口角色
- [ ] Children 章节完整
- [ ] 选择指南或决策树

## 建议项

- [ ] 共享资源目录（如适用）
- [ ] 子 Skill 简要说明
- [ ] @changelog.md

## 可选项

- [ ] Self-Reflection 使用统计
- [ ] 子 Skill 关系图
```

## 分支节点的特殊职责

### 路由职责

分支节点的核心职责是帮助选择正确的子 Skill：

```markdown
## 路由方式

1. **决策树** - 通过问题引导选择
2. **场景表** - 按场景列出推荐
3. **阶段指南** - 按开发阶段推荐
```

### 资源共享

分支节点可以管理子 Skill 共享的资源：

```markdown
## 共享资源管理

shared/
├── references/    # 所有子 Skill 可引用的参考资料
├── api-digests/   # 公共 API 摘要
├── checklists/    # 公共检查清单
└── templates/     # 公共模板
```

### 聚合能力

分支节点可以聚合子 Skill 的能力声明：

```markdown
## 聚合能力

此 Skill 树整体提供：
- 项目创建能力（via verseProjectInit）
- 架构设计能力（via verseArchitectureSelector）
- 组件开发能力（via verseComponent）
- ...
```
