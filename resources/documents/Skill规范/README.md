# Skill 规范文档

本文档定义了 Skill 系统的标准化规范，旨在支持自循环、自进化的 Agent 工作流。

## 核心理念

> **Skill = 上下文揭露节点**
>
> 以最高效的方法，揭示目标上下文。

Skill 系统采用 SceneGraph 节点树模型：
- 每个 Skill 是一个节点
- 节点可以有子节点（子 Skills）
- 父节点是对子节点能力的概括与编排
- 每个节点都有具体功能，包括"选择子 Skill"也是一种能力

## 快速开始

### 创建新 Skill

1. 阅读 [核心理念](基础规范/核心理念.md) 理解设计哲学
2. 参考 [节点结构](基础规范/节点结构.md) 创建目录
3. 按照 [章节模板](基础规范/章节模板.md) 编写 SKILL.md
4. 遵循 [命名规则](基础规范/命名规则.md) 确保一致性

### 理解 Agent 如何使用 Skill

1. 阅读 [全局注册表](基建规范/全局注册表.md) 了解发现机制
2. 阅读 [能力声明](基建规范/能力声明.md) 理解接口设计
3. 阅读 [执行协议](基建规范/执行协议.md) 理解执行流程

### 维护和代谢 Skill

1. 阅读 [代谢系统概述](代谢系统/README.md) 理解代谢回路
2. 阅读 [信号机制](代谢系统/信号机制.md) 了解如何产生和处理信号
3. 阅读 [代谢动作](代谢系统/代谢动作.md) 了解各类代谢操作
