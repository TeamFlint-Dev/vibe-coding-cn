## 如何添加新决策

1. 复制上方注释中的模板
2. 分配递增的 DR 编号
3. 填写所有字段（至少包含上下文、选项、决策、理由）
4. 更新顶部索引表
5. 提交并 `bd sync`

---

## 决策类别（参考）

记录决策时可参考以下类别：

| 类别 | 典型决策 |
|------|----------|
| 架构 | 选择哪种触发器、如何组织多个 workflow |
| 工具 | 使用哪个 MCP server、选择哪种存储方式 |
| 安全 | 权限范围、沙箱策略 |
| 流程 | 任务拆分方式、错误处理策略 |
| 集成 | 与外部系统的对接方式 |
