# 叙事设计技能

此目录用于存放与游戏叙事和剧情设计相关的技能，包括但不限于：

- 故事结构设计
- 角色塑造与对话
- 世界观构建
- 任务与支线设计
- 环境叙事

## 技能结构

待添加实际技能时，应遵循以下结构：

```
narrative/
├── SKILL.md          # 技能总述
├── shared/           # 共享资源
│   ├── references/   # 参考文档
│   └── templates/    # 模板文件
└── [subSkill]/       # 子技能目录
    └── SKILL.md
```
